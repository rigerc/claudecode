# Claude Code Development Plugin

Tools for developing and extending Claude Code with custom agents, commands, skills, and hooks.

## Commands

- `/create-agent` - Create a new Claude Code agent or sub-agent
- `/create-command` - Create a new Claude Code slash command
- `/create-hook` - Create a new Claude Code hook
- `/create-skill` - Create a new Claude Code Skill
- `/create-skill-from-documentation` - Create a skill from existing documentation

## Agents

- **meta-agent** - Generates new Claude Code sub-agent configuration files
- **mcp-expert** - Model Context Protocol (MCP) integration specialist
- **claude-optimizer** - Optimizes CLAUDE.md files for maximum effectiveness
- **context-manager** - Manages context across multiple agents and long-running tasks

## Skills

- **developing-claude-code-plugins** - Streamlined workflows for Claude Code plugin lifecycle
- **working-with-claude-code** - Complete official documentation for Claude Code features

## Installation

```bash
/plugin marketplace add ./plugins
/plugin install claude-code-development@plugins
```

## Use Cases

- Creating custom agents for specialized tasks
- Developing slash commands to automate workflows
- Building skills to extend Claude's capabilities
- Setting up hooks for event-driven automation
- Configuring MCP servers for external tool integration
