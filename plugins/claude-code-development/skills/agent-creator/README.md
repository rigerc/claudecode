# Agent Creator

Use when creating or updating Claude Code subagents. Provides expertise in agent configuration, system prompts, and workflow design.

## Structure

- `SKILL.md` - Main skill instructions
- `references/` - Detailed documentation loaded as needed
- `scripts/` - Executable code for deterministic operations
- `assets/` - Templates, images, or other resources

## Usage

This skill is automatically discovered by Claude when relevant to the task.
