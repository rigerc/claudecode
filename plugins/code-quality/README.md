# Code Quality Plugin

Code review, refactoring, and quality analysis tools for improving code maintainability and security.

## Commands

- `/code-review` - Comprehensive code quality review with security, performance, and architecture analysis
- `/refactor-code` - Refactor and improve code quality

## Agents

- **code-reviewer** - Expert code review specialist for quality, security, and maintainability analysis

## Installation

```bash
/plugin marketplace add ./plugins
/plugin install code-quality@plugins
```

## Use Cases

- Performing comprehensive code reviews with security analysis
- Refactoring legacy code for better maintainability
- Identifying performance bottlenecks and optimization opportunities
- Ensuring code follows best practices and architectural patterns
- Improving code readability and documentation quality
