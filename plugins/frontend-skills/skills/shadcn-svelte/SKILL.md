# shadcn-svelte

Frontend development skill for shadcn-svelte component library integration.

## When to Use
- Building Svelte/SvelteKit applications with shadcn-svelte
- Need component installation, customization, or best practices
- Form implementation with Superforms/Formsnap
- Theme setup and dark mode implementation
- Accessibility compliance and component optimization

## Quick Start
1. **Install component**: `npx shadcn-svelte@latest add button`
2. **Use in Svelte**: Import from `$lib/components/ui/button`
3. **Customize**: Modify component files directly in your project
4. **Theme**: Apply CSS variables for consistent design system

## Expert Guidance
- Component architecture and accessibility patterns
- Tailwind CSS integration and customization
- Form validation with SvelteKit and Superforms
- Performance optimization and bundle analysis
- TypeScript integration and type safety
- Testing strategies for component libraries

## Limitations
- Only for Svelte/SvelteKit projects (not React/Next.js)
- Requires Tailwind CSS setup
- Manual component management (not runtime dependencies)
