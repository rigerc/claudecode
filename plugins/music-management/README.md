# Music Management Plugin

Comprehensive music library management system guidance using the beets music organization tool.

## Skills

- **beets** - Comprehensive music library management system guidance covering setup, advanced querying, plugin development, and automation workflows for the beets music organization tool.

## Installation

```bash
/plugin marketplace add ./plugins
/plugin install music-management@plugins
```

## Use Cases

- Setting up and configuring beets for music library management
- Creating advanced queries to find specific music tracks
- Developing custom plugins for beets
- Automating music import and organization workflows
- Managing metadata and tags for large music collections
